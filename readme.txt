instalirati driver za neo4j: 

Tools -> NuGet Package Manager -> PAckage Manager console -> ispod u otvorenom prozoru odaberu projekt i rokni ovi komandu u terminal:
Install-Package Neo4j.Driver.Simple

Treba imati dignut lokalni neo4j na portu 7687 s userom neo4j i passwordom password (ovo sve je defaultno, port, user, pass)
